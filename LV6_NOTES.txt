1. NHAP FILE NGHIEP VU
- Ho tro doc file theo duong dan noi bo:
  + .csv
  + .txt
  + .json
  + .xls / .xlsx dang SpreadsheetML XML (file Excel 2003 XML)
- Co the goi qua /ask:
  doc file excel path: /duong/dan/file.xls va xuat excel
- Hoac qua API:
  POST /import-file
  body: { "filePath": "/duong/dan/file.csv", "exportExcel": true }
